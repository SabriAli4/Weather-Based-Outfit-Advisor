Weather-Based Outfit Advisor

Objective: Create a program that helps users decide what to wear based on the current weather conditions. The program should prompt the user for weather details and provide outfit suggestions accordingly.

Features:
User Input for Weather Conditions: Prompt the user to enter the current temperature, weather type (e.g., sunny, rainy, snowy), and wind speed.

Store Weather Details in Variables: Use appropriate variables to store the entered weather details.

Use Selection Statements to Determine Outfit: Use if-elif-else statements to suggest outfits based on the weather conditions.

Display Outfit Suggestion: Output the suggested outfit to the user.

Validate Input: Ensure all user inputs are valid before proceeding.

Detailed Implementation:
Initialize Variables for Storing Weather Details:
Create variables to store the temperature, weather type, and wind speed.

User Input for Weather Details:
Prompt the user to input the current temperature, weather type, and wind speed.
Validate the input to ensure it is within a reasonable range.

Use Selection Statements to Determine Outfit:
Use if-elif-else statements to decide on an outfit based on the weather conditions.
For example, if it's cold and windy, suggest a coat and scarf; if it's hot and sunny, suggest shorts and a t-shirt. Get creative!

Display Outfit Suggestion:

Print the outfit suggestion to the user.
